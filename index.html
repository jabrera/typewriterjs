Typewriter JS

About

Automatically types a text on an element

Prerequisite

  • jQuery

How to use

Call the function to create a new object

new TypeWriter(element, options);

You may also add some options depending on your preferences. options is in JSON format.

new TypeWriter($("#type"), {
		"data": ["Type this"]
		});

Options

  • data

    The text you want to display

    type Array
    values
    default ["Lorem ipsum"]

String Codes

  • >

    Make the animation type faster

  • <

    Make the animation type slower

  • |

    Pause the animation then continue after 1.5 seconds

Examples

Normal animation
Code 
new TypeWriter($("#sample1"), {})
Fast animation
Code 
new TypeWriter($("#sample3"), {
				"data": ["L>>>orem ipsum"]
			})
Slow animation
Code 
new TypeWriter($("#sample3"), {
				"data": ["L<<<orem ipsum"]
			})
Fast then slow animation
Code 
new TypeWriter($("#sample4"), {
				"data": ["L>>>orem <<<<<<ipsum"]
			})
